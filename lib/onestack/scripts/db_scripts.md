
## Onestack DB copy commands

### Copy from SERVER to LOCAL
`scp -i ~/.ssh/YOUR_KEY root@YOUR_SERVER_IP:/root/onestack_docker/DATABASE_FILE /path/to/local/destination/`

### Copy from LOCAL to SERVER
`scp -i ~/.ssh/YOUR_KEY /path/to/local/DATABASE_FILE root@YOUR_SERVER_IP:/root/onestack_docker/DATABASE_FILE`
